Belajar HTML dari Dasar Hingga Mahir

1. Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

<!DOCTYPE html>
<html>
<head>
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
</body>
</html>

2. Elemen Dasar HTML

  • <p> → Paragraf
  • <a> → Link
  • <img> → Gambar
  • <table> → Tabel
  • <form> → Formulir
